Ftd-SCRM旨在一站式解决移动互联网环境下的客户关系管理问题的解决方案。 

# 传统的CRM缺点
* 销售管理导向
* 对客户封闭
* 客户数据静止
* 商家对客户单向沟通
* 单个客户独立
* 客户无法自发成长。
 
FTD-SCRM这个项目本着解决传统CRM存在的诸多问题而诞生的，特点如下： 
* 用户关系导向
* 对客户开放
* 双向交互
* 客户之间相互关联
* 客户可自扩散。 

针对上述的特点，FTD-SCRM有以下几个大模块
* 用户获取
* 用户留存
* 精准营销
* 个性化服务
* 场景化电商

本项目是FTD-SCRM的服务器端，提供Open-API供前端调用， 待服务器端实现完毕后会跟进前端开发工作。
